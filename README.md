An interactive, professional-grade analytics dashboard built with **Streamlit** and **Plotly**, designed for fast data exploration and clean visualization.

---

## 🚀 Live Demo

👉 https://lowendcommitproject1.streamlit.app/

---

## ✨ Features

* 📈 Interactive time-series visualization
* 📊 Category-based aggregation charts
* 🎯 Dynamic filtering (date range & category)
* ⚡ Fast, cached data loading
* 📱 Responsive wide-layout dashboard
* 🧮 Real-time KPI metrics

---


## 🛠️ Tech Stack

* **Frontend**: Streamlit
* **Data Processing**: Pandas, NumPy
* **Visualization**: Plotly

---

## 📦 Installation

```bash
git clone https://github.com/yourusername/streamlit-analytics-dashboard.git
cd streamlit-analytics-dashboard
pip install -r requirements.txt
```

---

## ▶️ Run Locally

```bash
streamlit run app.py
```

---

## ☁️ Deployment

### Streamlit Cloud (Recommended)

1. Push this repo to GitHub
2. Go to https://streamlit.io/cloud
3. Click **New App**
4. Select your repo and branch
5. Set:

   * Main file: `app.py`
6. Deploy 🚀
